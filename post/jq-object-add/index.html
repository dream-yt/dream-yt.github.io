jq変形 {配列 => IDをキーとするオブジェクト}

Jul 3, 2019 15:30 · 181 words · 1 minute read

      • add

こんな配列を

[
  {"id": "123", "name": "abc"},
  {"id": "456", "name": "qwe"}
]

こうしたい

{
  "123": {"id": "123", "name": "abc"},
  "456": {"id": "456", "name": "qwe"}
}

add を使うとできる

$ pbpaste | jq '{ (.id): . } | [ . ] | add'
{
  "123": {
    "id": "123",
    "name": "abc"
  },
  "456": {
    "id": "456",
    "name": "qwe"
  }
}

add

通常は足し算や文字列結合に使われるが、 オブジェクトの配列に適用すると、それぞれの要素をマージしたオブジェクトができる

  • QUESTION: A way to concatenate input objects into one · Issue #70 · stedolan/jq
[
  { "123": {"id": "123", "name": "abc"} }
  { "456": {"id": "456", "name": "qwe"} }
]

  "123": {"id": "123", "name": "abc"},
