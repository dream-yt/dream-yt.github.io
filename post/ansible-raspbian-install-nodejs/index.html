[ansible x Raspbian] nodejs のインストール

Aug 28, 2020 00:07 · 65 words · 1 minute read

      • 環境

ansible-role-nodejs を使うとラズパイでも簡単に入る

  • geerlingguy/ansible-role-nodejs: Ansible Role - Node.js

一緒に npm -g install yarn もしてもらうような最小限の設定

  vars:
    nodejs_version: "12.x"
    nodejs_npm_global_packages:
      - yarn
  roles:
    - ansible-role-nodejs
