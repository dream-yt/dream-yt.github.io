perlでjsの分割代入 (Destructuring assignment) みたいなことをする

Mar 2, 2021 12:14 · 64 words · 1 minute read

  • 例

jsほどスマートじゃないが、それっぽいことはできる

例

こんなデータがあるとして

my $h = {
    a => 1,
    b => 2,
};

こうやって取り出す

my ($a, $b) = @$h{qw/a b/};
