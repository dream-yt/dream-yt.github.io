typescriptでのnullチェック

Jun 30, 2019 13:59 · 206 words · 1 minute read

  • 例
    • イコールふたつ
    • イコールみっつだと厳格
  • その他

基本的にはイコール2つの演算子でチェックすれば undefinedとnullのどちらかであればtrueになる

== null

例

$ ts-node
> const v = {a: 1, c: null}
undefined

イコールふたつ

nullとundefinedを区別しなくて楽

> v['b'] == null
true
> v['c'] == null
true

イコールみっつだと厳格

> v['c'] === null
true
> v['b'] === null
false

その他

宣言していない変数についても書いてあるが、これは特殊なコードになってる場合 普通のコードではイコール2つの演算子でチェックすればよい
