TypeScript で、関数の引数の型をtypeとして宣言したい場合

Apr 24, 2021 18:38 · 62 words · 1 minute read

Utility Type の Parameters を typeof と一緒に使えばよい。

const f = ({ a, b }) => {};
type FArg = Parameters<typeof f>;  //  type FArg = [{ a: any; b: any; }]

公式ドキュメントで紹介されている。
