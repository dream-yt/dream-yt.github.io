MySQLでWhere句に正規表現を使う

Feb 12, 2021 14:26 · 135 words · 1 minute read

    • 参考

こんな構文で正規表現で検索することができる

SELECT * FROM t WHERE url REGEXP 'https://.+:.+@.+'

ただし PCRE ではないので最低限のものしか使えない

正規表現が期待通りに書けてるかのテストはこんな感じでやるとよい

SELECT 'a' REGEXP '^[a-d]';
