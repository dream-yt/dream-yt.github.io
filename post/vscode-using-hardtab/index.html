VS Codeでインデントをハードタブにする

Nov 7, 2020 21:59 · 73 words · 1 minute read

この設定にするとTabキーでハードタブが挿入される

"editor.insertSpaces": false

たとえば(pep8違反だが)Pythonだけで適用したい場合は下記のように書く

    "settings": {
      "[python]": {
        "editor.insertSpaces": false,
    },
