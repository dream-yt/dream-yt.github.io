package.json の deps を深く考えずに全部upgradeする

Jan 11, 2020 16:34 · 461 words · 1 minute read

以下のコマンドでできる

$ npx npm-check-updates --upgrade

いちおうどんなアップグレードになるかは事前に確認できる

$ npx npm-check-updates

その他のオプションもいろいろある
